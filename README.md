# Generate React Monorepo Boilerplate

This repository contains the `generate-react-monorepo-boilerplate` Codex skill. It helps scaffold a production-ready React monorepo with a modern TypeScript toolchain and the usual pieces a real app team needs to move quickly.

## What It Sets Up

- React 18
- TypeScript
- MobX
- TanStack React Query
- TanStack Router
- Vite
- Vitest
- Cypress
- OpenAPI client scaffolding
- Workspace packages
- Documentation
- Git hooks
- CI

## When To Use It

Use this skill when you want to bootstrap:

- a React monorepo
- a multi-package TypeScript frontend workspace
- a larger SPA that needs shared packages and common tooling

## What You Get

The scaffold is designed to be a practical starting point for application work, with:

- a clean workspace layout
- shared app and tooling packages
- testing and quality checks from the start
- conventions that scale beyond a single package

## Typical Workflow

1. Run the boilerplate generator.
2. Customize package names, app structure, and tooling to fit your project.
3. Wire up APIs, routes, state management, and tests.
4. Extend CI and hooks as your team’s workflow matures.

## Notes

This repo is the skill implementation itself, not the generated app. If you are looking for the scaffold output, use the skill to create a new workspace in your target directory.
